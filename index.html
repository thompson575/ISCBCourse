Data Analysis with R and the Tidyverse


In advance of the course, please read the Introduction and ensure that you have R and RStudio installed on your laptop.


The introduction describes the R packages that will be covered and three key data analysis themes, reproducibility, legible coding and standardised workflow. These all introduced in the first session and are developed the subsequent sessions.


This website gives all of the materials for four half-day sessions, although the ISCB course will only cover session one. I hope that you will be motivated to work through the other sessions in your own time.


Each session consists of a lecture, a demonstration, an exercise and a solution and each is based on a different set of published data. The lecture introduces the R functions needed to analyse that data, the demonstration presents my analysis of the data and the exercise invites you to extend that analysis. The solutions should be treated as guides to ways in which you might tackle the exercises. There is no uniquely best way to analyse data.
